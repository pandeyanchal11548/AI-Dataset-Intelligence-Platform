# AI Dataset Intelligence Platform

## Project Overview

The AI Dataset Intelligence Platform is a Python project that analyzes datasets and generates reports for data quality, EDA, feature analysis, and machine learning readiness.

## Setup Guide

1. Install Python.
2. Install pandas and matplotlib.
3. Open the project in VS Code.
4. Run any Python file from the src folder.
